# Sushi Restaurant SQL Analytics Project

This project contains SQL analysis tasks built on a sushi restaurant dataset.  
The queries focus on real-world business analysis and advanced SQL concepts.

## Topics Covered
- Revenue & profit analysis
- Customer retention & churn
- JOIN operations
- CTEs
- Window functions
- Moving averages
- YoY & WoW growth
- Customer behavior analysis
- Menu performance analysis

## Tables Used
- `sushi_orders`
- `sushi_order_items`
- `sushi_customers`
- `sushi_reviews`
- `sushi_daily_pnl`
- `sushi_staff`

## SQL Skills Practiced
- Aggregations
- INNER/LEFT JOIN
- CASE WHEN
- Subqueries
- CTEs
- LAG()
- NTILE()
- Date functions

## Purpose
This project was created for SQL practice, portfolio building, and data analytics learning using realistic business scenarios.

## Dataset: https://www.kaggle.com/datasets/troykueh/sushi-restaurant-sales-simulation-10-years?select=sushi_orders.csv
